COME USARE L'AREA ADMIN

1. Copia questi 3 file nella root del progetto:
   - admin.html
   - admin.css
   - admin.js

2. Apri admin.html nel browser oppure pubblicalo insieme al sito.

3. Compila il form:
   - titolo
   - categoria
   - tempi
   - difficoltà
   - porzioni
   - ingredienti (uno per riga)
   - procedimento (uno per riga)
   - link fonte
   - immagine

4. Clicca "Genera JSON".

5. Clicca "Copia JSON" e incolla il blocco dentro recipes.json.

6. Copia la nuova immagine nella cartella images/.

7. Carica su GitHub:
   - recipes.json aggiornato
   - la nuova immagine

NOTA
L'immagine va indicata come:
images/nome-file.jpg
oppure solo:
nome-file.jpg
e verrà trasformata automaticamente.
