COME PUBBLICARLO SU GITHUB
1. Estrai lo ZIP.
2. Apri GitHub e crea un repository, per esempio: ricettario-neil
3. Carica tutti i file e le cartelle nella root del repository.
4. Vai in Settings > Pages
5. In Build and deployment imposta:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
6. Salva.
7. Attendi il link github.io.
8. Apri il link da Chrome su Android.
9. Tocca 'Installa app' oppure usa il menu di Chrome > Aggiungi a schermata Home / Installa app.
